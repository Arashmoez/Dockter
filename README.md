# dockter

`dockter` is a Python project designed to help with Docker-related workflows.

> `dockter` is a simple CLI tool for generating Dockerfiles interactively.

## Features

- Simple Python-based project structure
- Easy to install and run locally
- Designed for Docker/development workflow automation
- Configurable and extendable

## Requirements

- Python 3.10+
- pip
- Git

Clone the project in the root of your project then clone this repo and run
```python -m dockter```
